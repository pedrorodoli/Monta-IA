# MontañIA 🤖
Asistente inteligente avanzado diseñado para la gestión y apoyo en actividades de campamento, potenciado por IA Generativa.

## 💻 Características Técnicas
- **Core:** Basado en el modelo **Gemini 2.0/2.5** de Google para procesamiento de lenguaje natural.
- **Documentación:** Sistema de ingesta de archivos PDF mediante `pdf-parse` para consultas sobre normativas o guías.
- **Gestión:** Rotación inteligente de claves API para maximizar la disponibilidad del servicio.

## 🌐 Acceso
Disponible en: [https://montania.ddnsfree.com/](https://montania.ddnsfree.com/)

## 🛠️ Instalación
1. Configura tus `GEMINI_API_KEY` en el `.env`.
2. `npm install`
3. `npm start`
